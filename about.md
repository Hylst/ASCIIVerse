# À Propos d'ASCIIVerse Studio

**ASCIIVerse Studio** est une suite d'outils créatifs "frontend-only" conçue pour la génération de texte, l'art ASCII et le formatage pour les réseaux sociaux. 

## Notre Philosophie

- **Confidentialité Totale** : Toutes les transformations s'effectuent localement dans votre navigateur. Aucune donnée textuelle ou image n'est envoyée à un serveur (à l'exception des requêtes optionnelles à l'API Gemini pour la génération d'art ASCII).
- **Performance & Légèreté** : L'application est optimisée pour être rapide, réactive et compatible PWA (Progressive Web App) pour une utilisation hors-ligne.
- **Esthétique Terminal** : Une interface moderne inspirée du cyberpunk et des terminaux informatiques, utilisant un système de design cohérent et sombre.

## Les Outils Inclus

1. **Générateur ASCII** : Transformez vos textes en bannières ASCII massives avec des dizaines de polices.
2. **Image vers ASCII** : Convertissez vos images et photos en art ASCII détaillé.
3. **Studio Kaomoji** : Créez ou piochez dans une vaste bibliothèque d'émoticôes japonaises (e.g., `(╯°□°）╯︵ ┻━┻`).
4. **Formateur Social** : Préparez vos posts pour LinkedIn, Instagram ou Twitter avec des polices élégantes et une mise en page parfaite.
5. **Générateur de Tableaux** : Créez des tableaux en texte pur pour vos fichiers README ou vos commentaires de code.
6. **Canevas de Dessin** : Dessinez manuellement en utilisant des caractères ASCII.

## Technologie

ASCIIVerse est bâti avec les technologies web les plus modernes :
- **React 18** & **TypeScript** pour une logique robuste.
- **Vite** pour un build ultra-rapide.
- **Tailwind CSS** avec un système de variables sémantiques pour un thème dynamique.
- **Vite PWA** pour le support hors-ligne et l'installation sur mobile/desktop.
